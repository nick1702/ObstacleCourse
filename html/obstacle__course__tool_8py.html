Detailed Description

Obstacle Course Level Editor.

This file implements a PyQt5-based level editor for designing obstacle courses. It provides functionality for creating new courses and levels, importing and exporting course data as JSON, adding and modifying gates (with types and difficulty levels), and managing the graphical scene including grid drawing and gate interconnections.
